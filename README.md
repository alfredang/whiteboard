# Whiteboard

A minimal, browser-based drawing tool with a dark creative-studio aesthetic. Built for educators and trainers.

![Whiteboard Screenshot](screenshot.png)

## Features

- **Multi-page support** — add, delete, and navigate between pages with thumbnail preview
- **Pen & Eraser** tools with keyboard shortcuts (`B` / `E`)
- **Color picker** and adjustable brush size
- **Undo** support (Ctrl/Cmd+Z, up to 30 states per page)
- **Export PDF** — all pages exported as a single PDF document
- **Page navigation** — PgUp/PgDn or Ctrl+Arrow keys
- **Touch support** for tablets and mobile
- **Coordinate tracking** in the status bar
- Responsive layout with smooth animations

## Usage

Open `index.html` in any modern browser, or visit the live version:

**[Live Demo](https://alfredang.github.io/whiteboard/)**

## Tech

Pure HTML, CSS, and vanilla JavaScript. PDF export via [jsPDF](https://github.com/parallax/jsPDF).
