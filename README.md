# Whiteboard

A minimal, browser-based drawing tool with a dark creative-studio aesthetic.

![Whiteboard Screenshot](screenshot.png)

## Features

- **Pen & Eraser** tools with keyboard shortcuts (`B` / `E`)
- **Color picker** and adjustable brush size
- **Undo** support (Ctrl/Cmd+Z, up to 30 states)
- **Export** canvas as PNG
- **Touch support** for tablets and mobile
- **Coordinate tracking** in the status bar
- Responsive layout with smooth animations

## Usage

Open `index.html` in any modern browser, or visit the live version:

**[Live Demo](https://alfredang.github.io/whiteboard/)**

## Tech

Pure HTML, CSS, and vanilla JavaScript. No dependencies, no build step.
